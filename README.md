# Thera

A goal navigation system designed for neurodivergent minds. Not a task manager. Not a checklist app. A compass for exploration.

## Getting Started

```bash
npm install
cp .env.example .env.local
# Add your OpenAI API key to .env.local
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS v4
- Framer Motion
- React Flow
- Zustand (localStorage persistence)
- OpenAI API
